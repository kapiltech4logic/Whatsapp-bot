# MANAS WhatsApp Bot

A professional WhatsApp bot for the MANAS National Narcotics Helpline, providing anonymous reporting, rehabilitation support, and information on drug-related issues.

## Features

- **Anonymous Reporting**: Secure channels for reporting drug crimes.
- **Rehabilitation Guidance**: Support for addicts and counseling.
- **Interactive Menus**: User-friendly WhatsApp lists and buttons.
- **24/7 Availability**: Toll-free helpline integration.
- **Modular Architecture**: Clean, maintainable codebase.

## Prerequisites

- Node.js >= 18.0.0
- WhatsApp Business API account (for [PHONE_NUMBER_ID](http://_vscodecontentref_/22) and [WHATSAPP_TOKEN](http://_vscodecontentref_/23))
- Ngrok or similar for webhook tunneling (for local development)
